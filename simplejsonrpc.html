Simple JSON RPC Schema

Introduction

To complement Simple JSON Schema, Simple JSON RPC Schema is a way to specify a RPC end point using JSON. A Simple JSON RPC file should end in .sjrpc.

RPC URL Convention

Assuming an end point with a Simple JSON RPC Schema is at 

http://example.com/path/
then the Simple JSON RPC Schema should be available at 
http://example.com/path/?SimpleJSONRPCSchema

Simple JSON Schema

The Simple JSON Schema for a Simple JSON RPC file is: 

{ "name" : "string",
  "default_url" : "string",
  "description" : "string",
  "args" : [{ "name" : "string",
              "type" : "string,
              "default_value" : "string",
              "description" : "string"}],
  "return_type" : "string" }
